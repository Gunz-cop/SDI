# Prototipo SBI preservado

Este directorio conserva el prototipo preexistente identificado por la arquitectura como material de referencia no canónico. No forma parte de la compilación, lint, pruebas ni paquete activos de SDI.

La Etapa 1 no porta ni refactoriza este motor. Cualquier uso futuro debe respetar la arquitectura congelada y el roadmap de SDI 0.1.
